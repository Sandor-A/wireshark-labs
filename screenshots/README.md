
# Screenshots

This folder contains screenshots taken from Wireshark during network traffic analysis exercises.

The screenshots are used to visually support the analysis and findings documented in the repository, including:
- protocol hierarchy overview
- conversations and communication patterns
- DNS traffic inspection
- identification of non-standard ports
- basic indicators reviewed during SOC Level 1 investigations

## Purpose
The screenshots are intended to:
- provide visual evidence of analysis steps
- support documentation and reporting
- demonstrate practical use of Wireshark from a defensive (blue-team) perspective

## Notes
- All screenshots were taken from lab or training traffic
- No production or sensitive network data is included
- Screenshots are used strictly for educational and SOC Level 1 training purposes
